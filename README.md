# cns-china-notes

## 项目简介

本仓库用于归档和发布多个独立的 HTML 页面。这些页面以静态 HTML 文件的形式存储，不依赖于特定域名或后端服务。仓库主要作为内容存档与展示的集合，便于浏览和查阅。

## 目录说明

```
cns-china-notes/
├── README.md          # 本文件
├── index.html         # 入口页面（可选）
├── pages/             # 存放各独立 HTML 页面
│   ├── example1.html
│   ├── example2.html
│   └── ...
└── assets/            # 公共资源（CSS、JS、图片等）
    ├── style.css
    └── ...
```

- `pages/`：主要页面目录，每个 HTML 文件为一个独立页面。
- `assets/`：存放页面共用的样式、脚本或图片文件。

## 页面归档说明

- 所有页面均为纯静态 HTML，可直接在浏览器中打开运行。
- 页面间无强制依赖关系，可单独访问。
- 归档内容可能随时间更新或调整，不保证长期不变。
